# Protótipo - Svg Editor
https://diogoazvd.github.io/Svg-Editor/

## Funcionalidades
- Zoom e Pan
- Importar SVG (Somente Paths)
- Multi-seleção (Shift + Left Click)
- Transformações: Translate, Rotate, Scale (Com um elemento selecionado, pressione "T", "R", ou "S" para iniciar a operação)
- Durante a transformação, é possível aplicar valores utilizando o teclado
